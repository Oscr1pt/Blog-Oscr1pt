# Sistema de Catálogo y Carrito para Tienda de Calzado

Bien, como se puede ver en la imagen, la idea de este proyecto fue **poner en práctica lo aprendido** para desarrollar una plataforma web orientada a la **gestión y exhibición de productos** de una tienda de calzado. Para ello integré **PHP**, **MySQL** y una interfaz construida con **Skeleton CSS**.

El sistema permite mostrar productos de forma paginada, realizar búsquedas dinámicas y visualizar información detallada de cada artículo, incluyendo imágenes, precios y marcas.  
*(Nota: actualmente, al visitar el proyecto, no se pueden observar todas estas funciones debido a la falta de conexión con la base de datos).*

![login](/assets/img/HeaderCalzado.png)

Además, implementé un **carrito de compras interactivo** que permite agregar, modificar y vaciar productos en tiempo real mediante **JavaScript**, mejorando la experiencia del usuario y simulando el flujo básico de una tienda en línea.

También desarrollé un **módulo de administración** para la gestión del inventario, apoyándome en **AdminLTE** para facilitar la administración de usuarios y productos. Este módulo cuenta con un diseño limpio, funcional y adaptable a distintos dispositivos, pensado para un uso práctico y eficiente por parte del administrador.

![admin](/assets/img/shoesworld/adminLTE.png)

Este proyecto me permitió fortalecer habilidades en **arquitectura web**, **consultas a bases de datos**, **diseño responsivo** y, sobre todo, en la **integración entre frontend y backend** para crear experiencias completas y fluidas.
